### How To Run Locally
- Run mongod (MongoDB Daemon Process) by using the command

    - For macOS: `brew services start mongodb-community`

- Run mongosh (MongoDB shell terminal to connect on Compass)
    - `mongosh`

- Run the Backend API:
    - `npm run dev`
